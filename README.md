# Student Management System (Python)

This is my first Python project. it focuses on learning Object-Oriented Programming (OOP) and file handling.

## Features
- Add student
- View students
- Search student
- Update marks
- Delete student
- Data stored in file (students.txt)

## Concepts Used
- OOP (Classes & Objects)
- File Handling
- Lists and Data Management

## Project Structure
- main.py → user interaction (menu)
- student_manager.py → handles all operations
- students.py → defines Student class
